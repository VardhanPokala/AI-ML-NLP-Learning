# 10-Week AI/ML/NLP Learning Roadmap

## Week 1: Introduction to AI and ML
- **Day 1:**  
  - Theory: Overview of AI and ML, History, Applications  
  - Hands-on: Set up Python environment (Anaconda, Jupyter)  
  
- **Day 2:**  
  - Theory: Supervised vs. Unsupervised Learning  
  - Hands-on: Install libraries (NumPy, Pandas)  
  
- **Day 3:**  
  - Theory: Linear Regression Concept  
  - Hands-on: Implement Linear Regression with Scikit-learn  
  
- **Day 4:**  
  - Theory: Evaluation Metrics (MSE, R²)  
  - Hands-on: Evaluate Linear Regression Model  
  
- **Day 5:**  
  - Project: Build a Simple Linear Regression Model on a Dataset  
  
- **Days 6-7:**  
  - Review & Hands-on: Solve Python exercises on basic statistics.

## Week 2: Data Preprocessing and Visualization
- **Day 1:**  
  - Theory: Importance of Data Cleaning  
  - Hands-on: Handling Missing Values  
  
- **Day 2:**  
  - Theory: Feature Scaling (Normalization, Standardization)  
  - Hands-on: Apply Feature Scaling in Scikit-learn  
  
- **Day 3:**  
  - Theory: Exploratory Data Analysis (EDA)  
  - Hands-on: Visualize Data with Matplotlib and Seaborn  
  
- **Day 4:**  
  - Theory: Data Types and Transformations  
  - Hands-on: Data Type Conversion and Encoding Categorical Features  
  
- **Day 5:**  
  - Project: EDA on a real-world Dataset  
  
- **Days 6-7:**  
  - Review & Hands-on: Work on Data Cleaning exercises.

- **Days 6-7:**  
  - Review & Hands-on: Solve algorithmic challenges on Kaggle.
